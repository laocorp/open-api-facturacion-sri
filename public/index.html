Cómo funciona

Empezar es cuestión de minutos

Tres pasos y ya estás facturando electrónicamente con el SRI.

1

Regístrate y obtén tu API Key

Crea tu cuenta, configura tu emisor y recibe una clave sk_live_ segura para autenticar tus llamadas.

2

Envía tu comprobante

Mediante una llamada REST con los datos en JSON, tu factura, nota de crédito, retención o guía de remisión se genera, firma y envía al SRI.

3

Recibe tu PDF autorizado

El SRI autoriza tu comprobante y recibes el XML firmado más el PDF listo para entregar a tu cliente.

Tecnología

Construido para producción

Arquitectura robusta, segura y escalable. Pensada para negocios que necesitan confiabilidad.

API REST

Endpoints JSON con autenticación mediante API Key y JWT. Documentación Swagger interactiva.

SRI Ecuador

Compatible con los dos ambientes del SRI: certificación (pruebas) y producción. Firma XAdES-BES.

PDF y XML

Generación automática de PDF profesional. Almacenamiento seguro de XML firmados y autorizados.

Integración sencilla

Conéctate desde cualquier lenguaje: JavaScript, Python, PHP, Java o cualquier cliente HTTP.

Seguridad

API Key con hash bcrypt, rate limiting por tenant, y encriptación de datos sensibles. HTTPS obligatorio.

Documentación completa

Guía de inicio rápido, ejemplos en múltiples lenguajes, referencia de endpoints y códigos de error.
