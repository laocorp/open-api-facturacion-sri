Autenticación

Todos los endpoints de emisión requieren una API Key válida en el header:

X-Api-Key: sk_live_<tu_api_key>

Las API Keys se administran vía JWT en GET/POST /api-keys.

Administración (JWT)

Los endpoints de administración (API Keys, usage, configuración) usan JWT. Obtén tu token vía POST /auth/login.

Códigos de error

CódigoSignificadoSolución
400Datos inválidosRevisa el formato de los campos en Swagger
401API Key inválidaVerifica que el header X-Api-Key sea correcto
